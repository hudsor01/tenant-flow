Hook implementations:

4. `useBlogCategories()`: Spreads `blogQueries.categories()`. Simple wrapper.

5. `useRelatedPosts(category: string, excludeSlug: string, limit = 3)`: Spreads `blogQueries.related({ category, excludeSlug, limit })`. No keepPreviousData.

6. `useComparisonPosts(tag = 'comparison', limit = 6)`: Spreads `blogQueries.comparisons({ tag, limit })`. No keepPreviousData.
