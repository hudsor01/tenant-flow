Full rewrite of `src/app/blog/category/[category]/page.tsx`. Remove ALL hardcoded content: the `categoryConfig` map, icons, descriptions, raw blog card markup, raw inline newsletter. Replace with component composition using DB-resolved category names.

**Structure (top to bottom):**

1. **Data resolution** -- Core pattern (per locked decision: "display names resolved from database only via useBlogCategories()"):
   ```tsx
   const params = useParams()
   const categorySlug = params.category as string
   const router = useRouter()
   const [page] = useQueryState('page', parseAsInteger.withDefault(1))

   const { data: categories, isLoading: categoriesLoading } = useBlogCategories()
   const category = categories?.find(c => c.slug === categorySlug)

   // Redirect unknown slugs ONLY after categories finish loading
   useEffect(() => {
     if (!categoriesLoading && categories && !category) {
       router.replace('/blog')
     }
   }, [categoriesLoading, categories, category, router])

   // Use resolved display name for blog query
   const { data: blogData, isLoading: blogsLoading } = useBlogsByCategory(
     category?.name ?? '',
     page
   )
   ```
   CRITICAL: Do NOT redirect while `categoriesLoading === true`. This prevents flash redirects on valid category pages (Pitfall 1 from RESEARCH.md).
   CRITICAL: Pass `category.name` (display name) to `useBlogsByCategory`, NOT the URL slug (Pitfall 2 from RESEARCH.md).

2. **Back to Blog link** -- Same pattern as detail page.

3. **Category header** -- Per locked decision: "header shows name from DB + article count (no icons, no descriptions)":
   ```tsx
   <h1 className="text-3xl font-bold">{category.name}</h1>
   <p className="text-muted-foreground">{blogData?.total ?? 0} articles</p>
   ```
   No icon, no description, no color -- just name and count from DB.

4. **Loading state** -- Show `BlogLoadingSkeleton` while categoriesLoading or blogsLoading and category is valid.

5. **Empty state** -- Per locked decision and PAGE-05: "known categories with zero posts show BlogEmptyState component":
   ```tsx
   {category && !blogsLoading && blogData?.data.length === 0 && (
     <BlogEmptyState />
   )}
   ```

6. **Blog grid** -- Paginated grid of BlogCards with BlogPagination:

   ```tsx
   <div className="grid md:grid-cols-2 lg:grid-cols-3 gap-6">
     {blogData.data.map(post => <BlogCard key={post.id} post={post} />)}
   </div>
   <BlogPagination totalPages={blogData.totalPages} className="mt-8" />
   ```

7. **NewsletterSignup** -- Pre-footer section replacing raw inline newsletter:
   ```tsx
   <NewsletterSignup className="mt-16" />
   ```
   Per locked decision: "NewsletterSignup component as pre-footer section (replaces current raw inline newsletter)".
