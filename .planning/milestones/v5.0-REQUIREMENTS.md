# Requirements: TenantFlow — v5.0 AI Blog Content Engine (ARCHIVED)

**Defined:** 2026-06-07 · **Shipped:** 2026-06-10 · **9/9 requirements complete.**
**Core Value:** A local-LLM (LM Studio on the M5) + RAG n8n pipeline that drafts brand-positive, factually-grounded, E-E-A-T-credible blog posts into `n8n-blog-ingest` as `status='in-review'` drafts for human approval — and uses that engine to execute the SEO-01 reclaim. The bar: content that genuinely helps landlords and features TenantFlow naturally, never AI-spam listicles.

## v1 Requirements — all shipped

- [x] **BLOG-01**: n8n reaches LM Studio's OpenAI-compatible API (`/v1/chat/completions` + `/v1/embeddings` → 200). Resolved by running n8n NATIVELY on the Mac (node@22 + `n8n/start-native.sh`, `http://localhost:1234/v1`); colima's container→host network block made the Dockerized path unworkable.
- [x] **BLOG-02**: General-instruct model installed + smoke-tested — `mistral-small-3.2-24b-instruct-2506-mlx` (Apache 2.0) produces coherent on-brand prose; `qwen3-embedding` (dim 1024) + reranker reachable.
- [x] **BLOG-03**: TenantFlow fact corpus embedded via `qwen3-embedding` into pgvector (`blog_rag_chunks`, RLS-on, `match_blog_rag_chunks`); 10 chunks loaded + verified.
- [x] **BLOG-04**: `scripts/generate-blog-draft.ts` emits json_schema output passing all 9 ingest gates + the `validate_blog_post` trigger via a deterministic validate/repair loop + banlist sanitizer. Verified: a real 1,410-word, 8-H2, banlist-clean draft.
- [x] **BLOG-05**: HMAC-signs (`x-n8n-signature`) + POSTs to `n8n-blog-ingest` → real draft landed `status='in-review'` (HTTP 201, MCP-verified). EF migrated off the dead legacy service-role key to `INGEST_DB_KEY`; required `N8N_WEBHOOK_SECRET`.
- [x] **BLOG-06**: Quality + brand guardrails — E-E-A-T-hardened prompt + Organization "TenantFlow Team" byline (regression-locked) + a Mistral LLM-as-judge gate (4-dim score, regenerates/fails-closed on thin/off-brand drafts).
- [x] **BLOG-07**: Human approve/reject surface at `/admin/blog` (is_admin-walled): in-review list + sanitized preview → Approve (→published + revalidate) / Reject (→archived) via `blogs_update_admin` RLS + defense-in-depth `is_admin` re-check. Nothing publishes without an explicit Approve.
- [x] **BLOG-08**: SEO-01 reclaim — top-10 ghost-slug queue, generate-at-exact-slug (`--slug`), `reclaim-finalize` drops the redirect on publish + keeps the collision guard green. Closes the carried-over v4.0 SEO-01 item.
- [x] **BLOG-09**: Cadence + observability — pre-POST `public.blogs` slug-dedup (clean skip, no 409 waste), greppable `BLOG-GEN-FAIL` output, `EVERGREEN_TOPICS` secondary source, and documented sustainable few-posts/week cadence + Error-Trigger notify wiring (reuse `app_config`/`notify_critical_error`, no secret in source) + runaway/cost guards.
