CREATE OR REPLACE FUNCTION public.get_maintenance_stats(p_user_id uuid)
 RETURNS jsonb LANGUAGE plpgsql SECURITY DEFINER SET search_path TO 'public'
AS $function$
begin
  if p_user_id != (select auth.uid()) then
    raise exception 'Unauthorized';
  end if;
  return (
    select jsonb_build_object(
      'open', count(*) filter (where status = 'open'),
      ... ,
      'total', count(*)
    )
    from maintenance_requests
    where owner_user_id = p_user_id
  );
end;
$function$

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| authenticated client → SECURITY DEFINER RPC | The RPC runs as definer (bypasses RLS); the caller-supplied `p_user_id` is untrusted input crossing into definer context. |
| migration author → prod schema | Migration SQL grants EXECUTE; an over-broad grant (anon) would expose owner stats to unauthenticated callers. |

Proven by the dual-client RLS test in Plan 03. |
