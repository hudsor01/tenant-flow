## Core value
A local-LLM (LM Studio on the M5 Pro) + RAG n8n pipeline that drafts brand-positive, fact-grounded, E-E-A-T-credible blog posts into the `n8n-blog-ingest` Edge Function as `status='in-review'` drafts for human approval — and uses that engine to execute the SEO-01 ghost-slug reclaim. The bar: content that genuinely helps landlords and features TenantFlow naturally, never AI-spam listicles.

The full flow:
```
local Mistral-Small-3.2-24B + RAG (pgvector) → 9 ingest gates → LLM-as-judge (fail-closed)
   → status='in-review' → /admin/blog approve/reject → published
   → SEO-01 reclaim (--slug at exact deleted ghost URLs)
   → self-scheduled (few/week, reclaim-then-evergreen) + pre-POST dedup + BLOG-GEN-FAIL alerts
```

## Key accomplishments
1. **Local-LLM RAG generation pipeline** — native n8n (node@22) → LM Studio (`mistral-small-3.2-24b-instruct-2506-mlx`, Apache 2.0) + `qwen3-embedding` (dim 1024) into pgvector (`blog_rag_chunks` + `match_blog_rag_chunks` RPC). `scripts/generate-blog-draft.ts` produces contract-conforming drafts (9 gates + `validate_blog_post` trigger) via a deterministic validate/repair loop + banlist sanitizer; first real 1,410-word draft MCP-verified.
2. **Quality + brand guardrails** — an E-E-A-T-hardened prompt + a Mistral **LLM-as-judge** gate (4-dimension score, regenerate-bounded, **fails closed** — never POSTs a judge-rejected draft) catch thin/off-brand prose the deterministic gates can't.
3. **Human approve/reject surface** — `/admin/blog` in the `is_admin`-walled `(admin)` group: in-review list + sanitized markdown preview → Approve (→published + revalidate) / Reject (→archived) via `blogs_update_admin` RLS + a defense-in-depth `is_admin` re-check. Nothing publishes without an explicit Approve.
4. **SEO-01 reclaim** (closed the carried-over v4.0 item) — a `--slug` override regenerates a judge-gated post at the *exact* deleted high-impression ghost slug; a drift-guarded top-10 `RECLAIM_QUEUE`; and a safe `reclaim-finalize` codemod that drops the 301 + updates the collision guard on publish.
5. **Sustainable cadence + observability** — a pre-POST `public.blogs` slug-dedup (clean-skip, no wasted generation), greppable `BLOG-GEN-FAIL` output, an `EVERGREEN_TOPICS` secondary source, and documented few-posts/week cadence + Error-Trigger alerts + runaway/cost guards.
6. **The perfect-PR gate earned its keep** — every phase passed multi-cycle adversarial review; cycle-2 of the Phase-12 review caught and fixed a **live production incident** (a combined RLS policy referencing `is_admin()` raised `42501` on every anonymous `blogs` read, breaking `/blog`, sitemap, and feed for crawlers).
