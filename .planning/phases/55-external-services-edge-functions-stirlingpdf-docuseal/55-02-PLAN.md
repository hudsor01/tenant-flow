  <name>Task 1: Create docuseal outbound Edge Function</name>
  <files>supabase/functions/docuseal/index.ts</files>
  <action>
    Create supabase/functions/docuseal/index.ts. This handles outbound DocuSeal API calls: sending leases for signature, recording owner/tenant sign events, cancelling, and resending.

    Pattern: Follow stripe-connect Edge Function structure (authenticated endpoint, action-based dispatch).

    Authentication: JWT-authenticated (NOT public). Read Authorization header, verify with supabase.auth.getUser(token). Return 401 if missing/invalid.

    Environment variables:
    - DOCUSEAL_URL: base URL of self-hosted DocuSeal (e.g. https://sign.thehudsonfam.com)
    - DOCUSEAL_API_KEY: API token for DocuSeal API (X-Auth-Token header)
    - SUPABASE_URL, SUPABASE_SERVICE_ROLE_KEY: for auth verification

    Request format: POST with JSON body containing `action` field:

    **action: "send-for-signature"**
    Body: `{ action: 'send-for-signature', leaseId: string, message?: string, missingFields: { immediate_family_members: string, landlord_notice_address: string } }`

    Single approach (no alternatives): generate PDF via Edge Function → base64-encode → POST to DocuSeal `/api/submissions` with embedded document. Do NOT use DocuSeal template_id. Do NOT call `/api/submissions` without a document.

    Step-by-step:
    1. Look up the lease in Supabase: `supabase.from('leases').select('id, owner_user_id, primary_tenant_id, start_date, end_date, rent_amount, property_id, ...').eq('id', leaseId).single()`
    2. Look up owner email via `profiles` or `users` table using `lease.owner_user_id`
    3. Look up tenant email via `lease.primary_tenant_id → tenants.user_id → users/profiles table`
    4. Build a minimal HTML string (`leaseHtml`) containing the key lease fields: property address, tenant name, owner name, start date, end date, rent amount, and the `missingFields` values (immediate_family_members, landlord_notice_address).
    5. Call the generate-pdf Edge Function (server-to-server): `POST ${Deno.env.get('SUPABASE_URL')}/functions/v1/generate-pdf` with header `Authorization: Bearer ${Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')}` and body `{ html: leaseHtml, filename: 'lease-agreement.pdf' }`.
       - If generate-pdf returns non-2xx: immediately return 502 (fail-fast, no retry).
    6. Read the PDF response as ArrayBuffer, base64-encode it: `btoa(String.fromCharCode(...new Uint8Array(pdfBuffer)))`
    7. POST to DocuSeal: `${DOCUSEAL_URL}/api/submissions` with header `X-Auth-Token: ${DOCUSEAL_API_KEY}` and body:
       ```json
       {
         "documents": [{ "name": "Lease Agreement", "file": "data:application/pdf;base64,<base64Pdf>" }],
         "submitters": [
           { "role": "Property Owner", "email": "<ownerEmail>", "name": "<ownerName>", "order": 1 },
           { "role": "Tenant", "email": "<tenantEmail>", "name": "<tenantName>", "order": 2 }
         ],
         "send_email": true,
         "order": "preserved",
         "metadata": { "lease_id": "<leaseId>", "source": "tenantflow" }
       }
       ```
       - If `message` was provided in the request body, add `"message": "<message>"` to each submitter object.
       - If DocuSeal returns non-2xx: immediately return 502 (fail-fast, no retry).
    8. Extract `submission.id` from the DocuSeal response.
    9. Update Supabase via service role client: `leases set sent_for_signature_at = now(), docuseal_submission_id = String(submission.id) where id = leaseId`
    10. Return: `{ success: true, submission_id: submission.id }`

    **action: "sign-owner"**
    Body: `{ action: 'sign-owner', leaseId: string }`
    - Look up lease.docuseal_submission_id from Supabase
    - GET `${DOCUSEAL_URL}/api/submitters?submission_id=${submissionId}` to find the owner submitter
    - Update leases.owner_signed_at = now(), owner_signature_method = 'docuseal' in Supabase service role client
    - Return: `{ success: true }`

    **action: "sign-tenant"**
    Body: `{ action: 'sign-tenant', leaseId: string }`
    - Same as sign-owner but for tenant
    - Update leases.tenant_signed_at = now(), tenant_signature_method = 'docuseal'
    - If both owner_signed_at and tenant_signed_at are set: update lease_status to 'active'
    - Return: `{ success: true }`

    **action: "cancel"**
    Body: `{ action: 'cancel', leaseId: string }`
    - Look up lease.docuseal_submission_id
    - POST `${DOCUSEAL_URL}/api/submissions/${submissionId}/archive`
    - Update leases: docuseal_submission_id = null, sent_for_signature_at = null, lease_status = 'draft'
    - Return: `{ success: true }`

    **action: "resend"**
    Body: `{ action: 'resend', leaseId: string, message?: string }`
    - Look up lease.docuseal_submission_id
    - Get submitters, find pending ones, PUT `${DOCUSEAL_URL}/api/submitters/${submitterId}` with { send_email: true, message? }
    - Return: `{ success: true }`

    All DocuSeal API calls:
    - Use X-Auth-Token: ${DOCUSEAL_API_KEY} header
    - Content-Type: application/json
    - No retry (fail-fast — matching stripe-webhooks pattern per locked decision)
    - On non-2xx response: return 502 with error message

    Use the service role Supabase client for ALL DB operations (reads and writes) — not the user JWT client. This is an Edge Function with elevated privileges for updating lease records.

    CORS headers:
    ```typescript
    const corsHeaders = {
      'Access-Control-Allow-Origin': '*',
      'Access-Control-Allow-Headers': 'authorization, x-client-info, apikey, content-type',
    }
    ```

    Handle OPTIONS preflight.

    ```typescript
    async function callDocuSealEdgeFunction(
      action: string,
      payload: Record<string, unknown>
    ): Promise<{ success: boolean }> {
      const supabase = createClient()
      const { data: sessionData } = await supabase.auth.getSession()
      const token = sessionData.session?.access_token
      if (!token) throw new Error('Not authenticated')

      const baseUrl = process.env.NEXT_PUBLIC_SUPABASE_URL
      const response = await fetch(`${baseUrl}/functions/v1/docuseal`, {
        method: 'POST',
        headers: {
          Authorization: `Bearer ${token}`,
          'Content-Type': 'application/json',
        },
        body: JSON.stringify({ action, ...payload }),
      })

      if (!response.ok) {
        const error = await response.json().catch(() => ({ error: response.statusText }))
        throw new Error((error as { error?: string }).error ?? 'DocuSeal request failed')
      }

      return response.json()
    }
    ```
