| pre-flip migration → live enforcement | the flag flip is the irreversible enable; ordering guarantees the grandfather snapshot exists before any block |
| grandfather stamp → users.storage_grandfathered_at | only the migration (service_role) sets it — a user must not self-grandfather |
| Max-tier owners → snapshot | Max/unlimited owners must NOT be stamped (they are never enforced anyway; stamping would be noise) |
