**1. Create shared ErrorPage component**

Create `src/components/shared/error-page.tsx` -- a 'use client' component with:
- AlertCircle icon from lucide-react (size-12, text-destructive)
- "Something went wrong" heading
- "An unexpected error occurred. Please try again." description in text-muted-foreground
- TWO escape actions per locked decision:
  - "Try Again" button (calls resetAction/reset prop)
  - "Go to Dashboard" button (Link to configurable dashboardHref, default "/dashboard")
- Both buttons use shadcn Button with variant="outline" and size="sm"
- RefreshCw icon on retry, Home icon on dashboard link
- Sentry error capture via useEffect
- Accept props: `{ error: Error & { digest?: string }, resetAction: () => void, dashboardHref?: string }`

Error detail level is Claude's discretion per CONTEXT.md. For simplicity, show the generic message. Do NOT expose error.message to the user (security practice from CLAUDE.md).
