<interfaces>
<!-- Edge Function details -->
stripe-autopay-charge key behaviors:
- Called by pg_cron (service_role) to process automatic rent payments
- Requires service_role auth or specific authorization
- Looks up rent_due records where autopay is enabled and due_date <= current_date
- Creates Stripe payment intents with idempotency keys (rent_due_id + tenant_id)
- Handles retry logic (3 attempts over 7 days)
- Parallelizes independent DB queries via Promise.all()
- Returns count of processed/failed charges

tenant-invitation-accept key behaviors:
- Unauthenticated endpoint (but requires JWT for tenant identity)
- Rate limited: 10 req/min per IP (rateLimit() from _shared/rate-limit.ts)
- Accepts invitation code from request body
- Validates invitation exists and hasn't expired
- Links tenant to property/lease
- Auth pattern: extract Bearer token, then auth.getUser(token)

Test infrastructure:
- Deno test runner in supabase/functions/tests/
- createClient from npm:@supabase/supabase-js@2
- jsr:@std/assert for assertions
- jsr:@std/dotenv/load for env vars
</interfaces>
