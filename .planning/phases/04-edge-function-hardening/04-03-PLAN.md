For each of these 6 Edge Functions, apply two changes:

**A. Add env validation** (EDGE-01) -- at the top of the `Deno.serve` handler, after CORS preflight, add:
```typescript
import { validateEnv } from '../_shared/env.ts'
import { errorResponse } from '../_shared/errors.ts'

// Inside Deno.serve handler, after handleCorsOptions:
let env: Record<string, string>
try {
  env = validateEnv({
    required: [/* function-specific list */],
    optional: [/* function-specific list */],
  })
} catch (err) {
  return errorResponse(req, 500, err, { action: 'env_validation' })
}
```
Then use `env.STRIPE_SECRET_KEY` etc. instead of `Deno.env.get('STRIPE_SECRET_KEY')!` throughout.
