async function sendReceiptEmails(

**Inside `sendReceiptEmails`, implement this flow:**

1. **Extract metadata** from `pi.metadata`:
   - `tenant_id`, `lease_id`, `property_id`, `unit_id`
   - `amount`, `period_month`, `period_year`

2. **Resolve data in parallel** (4 queries):
   ```typescript
   const [tenantResult, leaseResult, propertyResult, unitResult] = await Promise.all([
     supabase.from('tenants').select('user_id').eq('id', metadata.tenant_id).single(),
     supabase.from('leases').select('owner_user_id').eq('id', metadata.lease_id).single(),
     supabase.from('properties').select('address_line1, city, state, postal_code, name').eq('id', metadata.property_id).single(),
     supabase.from('units').select('unit_number').eq('id', metadata.unit_id).single(),
   ])
   ```

3. **Bail early** if any critical lookup failed (log error, return -- do not throw):
   ```typescript
   if (!tenantResult.data || !leaseResult.data || !propertyResult.data) {
     console.error('[RESEND_ERROR] Missing critical data for receipt emails:', {
       tenant: !!tenantResult.data, lease: !!leaseResult.data, property: !!propertyResult.data
     })
     return
   }
   ```

4. **Resolve user emails and names in parallel:**
   ```typescript
   const [tenantUserResult, ownerUserResult] = await Promise.all([
     supabase.from('users').select('email, full_name').eq('id', tenantResult.data.user_id).single(),
     supabase.from('users').select('email, full_name').eq('id', leaseResult.data.owner_user_id).single(),
   ])
   ```

5. **Check notification_settings for both users in parallel:**
   ```typescript
   const [tenantSettingsResult, ownerSettingsResult] = await Promise.all([
     supabase.from('notification_settings').select('email').eq('user_id', tenantResult.data.user_id).maybeSingle(),
     supabase.from('notification_settings').select('email').eq('user_id', leaseResult.data.owner_user_id).maybeSingle(),
   ])
   ```
   - If `notification_settings` row exists AND `email === false`, skip that user's email
   - If no row exists (null result), DEFAULT TO SENDING (absence = opt-in)
   - Log skipped emails: `console.log('[EMAIL_SKIP] User {id} has email notifications disabled')`

6. **Resolve payment method last 4 digits** from the charge already retrieved earlier in the handler, OR via `pi.payment_method`:
   - First check if the `charge` object (from the balance_transaction expansion) has `payment_method_details?.card?.last4`
   - If not available, try `stripe.paymentMethods.retrieve(pi.payment_method as string)` and get `card?.last4`
   - Wrap in try-catch; if fails, set `paymentMethodLast4 = null` (non-critical field)

7. **Format data** for templates:
   - `amount`: format as `$X,XXX.XX` using `(grossAmount).toLocaleString('en-US', { style: 'currency', currency: 'USD' })`
   - `propertyAddress`: `${address_line1}, ${city}, ${state} ${postal_code}`
   - `paymentDate`: format current date as readable string (e.g., "February 27, 2026")
   - `lateFeeAmount`: format if `rentPaymentRecord.late_fee_amount` is not null
   - `baseRentAmount`: calculate as `grossAmount - lateFeeAmount` if late fee present

8. **Render and send emails in parallel using Promise.allSettled:**
   ```typescript
   const emailPromises: Promise<void>[] = []

   // Tenant receipt
   if (tenantUserResult.data?.email && tenantEmailEnabled) {
     emailPromises.push((async () => {
       const html = await renderAsync(
         React.createElement(PaymentReceipt, { /* props */ })
       )
       const result = await sendEmail({
         to: [tenantUserResult.data.email],
         subject: `Payment Receipt - ${metadata.period_month} ${metadata.period_year}`,
         html,
         tags: [{ name: 'type', value: 'payment_receipt' }, { name: 'tenant_id', value: metadata.tenant_id }],
       })
       if (!result.success) {
         console.error('[RESEND_ERROR] Tenant receipt failed:', result.error)
       }
     })())
   }

   // Owner notification
   if (ownerUserResult.data?.email && ownerEmailEnabled) {
     emailPromises.push((async () => {
       const html = await renderAsync(
         React.createElement(OwnerNotification, { /* props */ })
       )
       const result = await sendEmail({
         to: [ownerUserResult.data.email],
         subject: `Payment Received - ${tenantUserResult.data?.full_name ?? 'Tenant'}`,
         html,
         tags: [{ name: 'type', value: 'owner_payment_notification' }, { name: 'lease_id', value: metadata.lease_id }],
       })
       if (!result.success) {
         console.error('[RESEND_ERROR] Owner notification failed:', result.error)
       }
     })())
   }

   await Promise.allSettled(emailPromises)
   ```
