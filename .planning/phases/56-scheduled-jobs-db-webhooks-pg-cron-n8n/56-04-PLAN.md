**n8n workflow JSON structure (v1.x format):**
```json
{
  "name": "Workflow Name",
  "nodes": [...],
  "connections": { "NodeName": { "main": [[{ "node": "NextNodeName", "type": "main", "index": 0 }]] } },
  "active": false,
  "settings": { "executionOrder": "v1" }
}
```
