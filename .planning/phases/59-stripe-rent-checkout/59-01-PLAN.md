```sql
create index if not exists idx_rent_payments_rent_due_id on rent_payments(rent_due_id);
```

```sql
create unique index if not exists idx_rent_payments_rent_due_unique_succeeded
  on rent_payments(rent_due_id)
  where status = 'succeeded' and rent_due_id is not null;
```

<task type="auto">
  <name>Task 2: Create stripe-rent-checkout Edge Function</name>
  <files>supabase/functions/stripe-rent-checkout/index.ts</files>
  <action>
Create a new Supabase Edge Function at `supabase/functions/stripe-rent-checkout/index.ts`.

**Purpose:** Creates a Stripe Checkout Session in `payment` mode with a destination charge routing funds to the owner's Stripe Express account.

**Request:** POST with JSON body `{ rent_due_id: string }`. Authenticated via JWT Bearer token (tenant user).

**Implementation flow:**

1. **CORS + Auth** — Use the same pattern as stripe-checkout/index.ts:
   - Import and use `getCorsHeaders` and `handleCorsOptions` from `../_shared/cors.ts`
   - Verify JWT via `supabase.auth.getUser(token)`

2. **Resolve tenant** — Query `tenants` table where `user_id = user.id` to get `tenant_id`. Return 404 if not found.

3. **Validate rent_due** — Query `rent_due` by `rent_due_id` from request body. Return 400 if not found.

4. **Duplicate payment check** — Query `rent_payments` where `rent_due_id = rent_due_id` and `status = 'succeeded'`. If found, return 409 with `{ error: 'Already paid for this period' }`.

5. **Resolve lease** — From `rent_due.lease_id`, query `leases` to get `owner_user_id`, `rent_amount`, `unit_id`. Also verify tenant has access via `lease_tenants` where `lease_id` and `tenant_id` match. Return 403 if tenant is not on this lease.

6. **Resolve connected account** — Query `stripe_connected_accounts` where `user_id = owner_user_id`. Return 400 with `{ error: 'Property owner has not set up payment receiving. Please contact your landlord.' }` if not found.

7. **Check charges_enabled** — If `charges_enabled` is false/null, return 400 with `{ error: 'Property owner payment setup is incomplete. Please contact your landlord.' }`.

8. **Calculate fees** — Per CONTEXT.md, owner absorbs all fees (tenant pays exact rent amount):
   - `amount_cents = rent_due.amount * 100` (rent_due.amount is in dollars)
   - `platform_fee_percent = connected_account.default_platform_fee_percent ?? 5` (default 5%)
   - `application_fee_cents = Math.round(amount_cents * platform_fee_percent / 100)`

9. **Resolve property_id** — Query `units` where `id = lease.unit_id` to get `property_id`.

10. **Create Stripe Checkout Session:**
```typescript
const session = await stripe.checkout.sessions.create({
  mode: 'payment',
  payment_method_types: ['card'],
  line_items: [{
    price_data: {
      currency: 'usd',
      product_data: {
        name: 'Rent Payment',
        description: `Rent for ${periodMonth}/${periodYear}`,
      },
      unit_amount: amount_cents,
    },
    quantity: 1,
  }],
  payment_intent_data: {
    application_fee_amount: application_fee_cents,
    transfer_data: {
      destination: connected_account.stripe_account_id,
    },
    metadata: {
      tenant_id: tenant.id,
      lease_id: lease.id,
      property_id: property_id,
      unit_id: lease.unit_id,
      rent_due_id: rent_due_id,
      amount: String(rent_due.amount),
      period_month: String(new Date(rent_due.due_date).getMonth() + 1),
      period_year: String(new Date(rent_due.due_date).getFullYear()),
    },
  },
  success_url: `${frontendUrl}/tenant?checkout=success`,
  cancel_url: `${frontendUrl}/tenant?checkout=cancelled`,
  expires_after: 1800, // 30 minutes
})
```

11. **Return** `{ url: session.url, session_id: session.id }` with 200.

**Error handling:**
- All DB errors return 500 with generic message
- Stripe errors return 502 with `{ error: 'Payment service unavailable. Please try again.' }`
- Log errors to console for Edge Function logs

**Environment variables used:**
- `STRIPE_SECRET_KEY`
- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`
- `FRONTEND_URL`

**Important:** Use `Stripe` from the import map (`stripe@14.25.0`). Use `apiVersion: '2024-06-20'` to match existing Edge Functions. Use `createClient` from `@supabase/supabase-js` import map.
