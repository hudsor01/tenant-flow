# Plan 10-01: Remove Dead Code and Orphaned Components

**Goal:** Remove all dead code and orphaned components identified by the v1.0 milestone audit integration checker.

## Context

The milestone audit integration checker found:
1. Dead monitoring rate limiter in proxy.ts (lines 8-55, 113-117) — `config.matcher` excludes `/monitoring`, so this code never executes
2. Dead Zod schemas in `src/shared/validation/auth.ts` — `authResponseZodSchema` and `userProfileResponseZodSchema` are never imported
3. Orphaned `EmptyState` component — created in Phase 7 but never imported; list pages use `Empty` compound directly
4. Orphaned `VirtualizedList` component — created in Phase 8 but superseded by direct `useVirtualizer` in table components

## Task 1: Remove dead code from proxy.ts and auth.ts

**Files to modify:**
- `proxy.ts`
- `src/shared/validation/auth.ts`
- `src/shared/validation/__tests__/auth.test.ts`

**Steps:**

1. In `proxy.ts`:
   - Remove the monitoring rate limiter constant block (lines 8-11): `MONITORING_RATE_LIMIT`, `MONITORING_WINDOW_MS`, `monitoringRateMap`, `monitoringRequestCount`
   - Remove the `getMonitoringIp()` function (lines 13-21)
   - Remove the `isMonitoringRateLimited()` function (lines 23-55)
   - Remove the monitoring rate limit check in the `proxy()` function (lines 113-117): the `if (pathname === '/monitoring' ...)` block
   - Keep the comment block above (line 7 "// ---") only if needed for separation

2. In `src/shared/validation/auth.ts`:
   - Remove `authResponseZodSchema` export (starts ~line 68)
   - Remove `userProfileResponseZodSchema` export (starts ~line 87)
   - Keep all other schemas (loginZodSchema, registerZodSchema, signupFormSchema, etc.)

3. In `src/shared/validation/__tests__/auth.test.ts`:
   - Remove the dead code comment referencing these schemas (~line 83)

**Verification:**
```bash
pnpm typecheck && pnpm lint
grep -r "authResponseZodSchema\|userProfileResponseZodSchema" src/ --include="*.ts" --include="*.tsx"
grep -r "isMonitoringRateLimited\|monitoringRateMap\|getMonitoringIp" . --include="*.ts" --include="*.tsx"
```
