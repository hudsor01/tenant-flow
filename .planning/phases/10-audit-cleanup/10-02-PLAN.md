## Task 1: Fix CLAUDE.md inaccuracies

**File to modify:** `CLAUDE.md`

**Steps:**

1. In the "Query Key Factories" section, find the factory files list. Remove the line:
   ```
   - `dashboard-keys.ts` — owner dashboard stats, portfolio data
   ```
   Dashboard keys are defined inline in `src/hooks/api/use-owner-dashboard.ts`, not in a separate factory file. Do NOT add a replacement line — only factory files in `src/hooks/api/query-keys/` should be listed.

2. In the "Performance Conventions" section, remove the line about VirtualizedList:
   ```
   - List views use `VirtualizedList` from `#components/shared/virtualized-list` with fixed row heights and overscan of 5
   ```
   Replace with a note about direct useVirtualizer usage:
   ```
   - Table list views use `useVirtualizer` from `@tanstack/react-virtual` directly on tbody rows
   ```

3. In the "Component Conventions" section, remove the line about EmptyState:
   ```
   - Use `EmptyState` from `#components/shared/empty-state` for list page empty states
   ```
   Replace with:
   ```
   - Use `Empty` compound component from `#components/ui/empty` for list page empty states
   ```

4. In the "Performance Conventions" section, find the VirtualizedList scroll container note and remove it:
   ```
   - `VirtualizedList` scroll container needs explicit height (`h-[calc(100vh-Xpx)]`) -- without height, all items render
   ```
