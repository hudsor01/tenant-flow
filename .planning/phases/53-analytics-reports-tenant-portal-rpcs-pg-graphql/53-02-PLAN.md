- `useExpenseSummary()` → `supabase.rpc('get_expense_summary', { p_user_id: userId })` — shape into `ExpenseSummaryData`.

**Financial statements hooks (income statement, cash flow, balance sheet):**
These don't have specific RPCs. Use `get_financial_overview` and shape the data:
- `useIncomeStatement(params)` → `supabase.rpc('get_financial_overview', { p_user_id: userId })` — shape into `ApiResponse<IncomeStatementData>` with income/expense line items derived from the overview data. **Expense hooks:**
- `useExpenses()` → `supabase.from('expenses').select('id, description, category, amount, property_id, expense_date, vendor_name, maintenance_request_id, created_at').order('expense_date', { ascending: false })`. If `expenses` table doesn't exist, return empty array stub with comment.
