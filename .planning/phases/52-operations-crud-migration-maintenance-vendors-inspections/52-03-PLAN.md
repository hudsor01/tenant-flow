Create three RLS isolation test files following the exact same pattern as `apps/integration-tests/src/rls/properties.rls.test.ts` (read that file first).

The cross-tenant isolation pattern is:
```
beforeAll: authenticate ownerA + ownerB clients, capture their user IDs
test 1: ownerA queries table → all rows have owner_user_id = ownerAId
test 2: ownerB queries table → all rows have owner_user_id = ownerBId
test 3: cross-tenant check → ownerA's result IDs and ownerB's result IDs have no overlap
afterAll: sign out both clients
```

**`apps/integration-tests/src/rls/maintenance.rls.test.ts`:**
- Query: `supabase.from('maintenance_requests').select('id, owner_user_id')`

- Verify: all rows returned by ownerA have `owner_user_id === ownerAId`, vice versa for ownerB
