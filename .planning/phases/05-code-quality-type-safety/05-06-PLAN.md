**Systematic 'use client' audit of ALL 494 files (CODE-15):**

Per user decision: "Full audit of all 494 'use client' files (not just page files)" — this is a locked decision, not best-effort.

6. **Categorize all 494 files** by running:
   ```bash
   grep -rl "'use client'" src/ | sort
   ```
   Organize files into these categories for systematic review:

   **Category A — Page files (~63 files):**
   Files matching `src/app/**/page.tsx`. For each:
   - Does the page itself use hooks (useState, useEffect, custom hooks)? -> Must stay client
   - Does it only compose client children without using hooks? -> Remove directive, let children carry it
   - Does it export `generateMetadata` or `generateStaticParams`? -> Must be Server Component (remove directive)

   **Category B — Layout files:**
   Files matching `src/app/**/layout.tsx`. Layouts should almost always be Server Components. Remove directive unless they use hooks directly.

   **Category C — Component files (src/components/**):**
   - Pure display components (only receive props, render JSX) -> Remove directive
   - Components that import client components but don't use hooks -> Remove directive (the import boundary handles it)
   - Components using hooks or event handlers -> Must stay client

   **Category D — Hook files (src/hooks/**):**
   These always need 'use client' (they use React hooks by definition). Skip.

   **Category E — Type/utility/constant files:**
   Files in `src/shared/types/`, `src/lib/`, `src/utils/` — should NOT have 'use client'. Remove all instances.

   **Category F — Provider files (src/components/providers/*, src/providers/*):**
   Providers use React context (useContext, createContext) — must stay client. Skip.

7. **For each file where directive is removed:**
   - Verify the file does not directly use any client-only APIs (hooks, event handlers, browser APIs)
   - If removing the directive breaks typecheck, put it back
   - Extract interactive parts to a child client component if the parent can become a Server Component

8. **Track results**: Count files before and after. Report:
   - Total files with 'use client' before: 494
