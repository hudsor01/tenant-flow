Critical memory from MEMORY.md:
- SUPABASE_PUBLISHABLE_KEY=sb_publishable_* is NOT a JWT. supabase-js overrides global.headers.Authorization internally, so auth.uid() = null in RLS. Use the anon key (JWT) for the browser client so auth.uid() works.
- JWT cache key collision: All Supabase JWTs share identical first 16 chars (HS256 header). Use sig.slice(-32) as cache key.
