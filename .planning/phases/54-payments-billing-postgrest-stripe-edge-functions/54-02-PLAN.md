**Import Stripe via npm specifier (Deno-compatible):**
```typescript
import Stripe from 'npm:stripe@14'
```

**Authentication pattern (same as export-report):**
- Read `Authorization: Bearer <token>` header
- Verify via `supabase.auth.getUser(token)`
- Return 401 if missing or invalid

**Actions via request body JSON `{ action, ... }`:**

1. `action: 'account'` (GET-like, POST body) — returns account status:
   - Query `stripe_connected_accounts` table WHERE user_id = auth.uid()
   - If no row: return `{ account: null, hasAccount: false }`
   - If row exists: call `stripe.accounts.retrieve(stripe_account_id)` to get live status
   - Map to `ConnectedAccountWithIdentity` shape: charges_enabled, payouts_enabled, requirements_due, onboarding_status
   - On Stripe API error: return DB data only (don't break if Stripe unreachable)

2. `action: 'onboard'` — creates/refreshes onboarding URL:
   - Input: `{ displayName?, businessName?, country?, entityType? }` from request body
   - Query stripe_connected_accounts WHERE user_id = user.id
   - If no account: call `stripe.accounts.create({ type: 'express', country: country ?? 'US', email: user.email, capabilities: { card_payments: { requested: true }, transfers: { requested: true } } })`
   - Insert into stripe_connected_accounts: `{ user_id, stripe_account_id: account.id, business_type: entityType ?? 'individual', onboarding_status: 'pending', business_name: businessName ?? null }`
   - Then create Account Link: `stripe.accountLinks.create({ account: stripe_account_id, refresh_url: returnUrl + '/settings/payouts', return_url: returnUrl + '/dashboard?stripe_connect=success', type: 'account_onboarding' })`
   - Return `{ onboardingUrl: accountLink.url, accountId: account.id }`
   - If account already exists: skip create, go directly to accountLinks.create with existing stripe_account_id

3. `action: 'refresh-link'` — refreshes expired onboarding link:
   - Get stripe_connected_account for user
   - Call `stripe.accountLinks.create(...)` with same params
   - Return `{ onboardingUrl: accountLink.url }`

4. `action: 'balance'` — get connected account balance:
   - Get stripe_connected_account for user
   - Call `stripe.balance.retrieve({ stripeAccount: stripe_account_id })`
   - Return `{ balance: { available, pending } }`

5. `action: 'payouts'` — list payouts:
   - Input: `{ limit?, starting_after? }`
   - Call `stripe.payouts.list({ limit, starting_after }, { stripeAccount: stripe_account_id })`
   - Return `{ payouts: data, hasMore: has_more }`

6. `action: 'transfers'` — list transfers:
   - Call `stripe.transfers.list({ destination: stripe_account_id, limit, starting_after })`
   - Return `{ transfers: data, hasMore: has_more }`

**CORS headers** (same pattern as export-report):
```typescript
const corsHeaders = {
  'Access-Control-Allow-Origin': '*',
  'Access-Control-Allow-Headers': 'authorization, x-client-info, apikey, content-type',
}
```
Handle OPTIONS preflight with `return new Response('ok', { headers: corsHeaders })`

**Error handling:**
- Stripe API errors: return 500 with `{ error: stripeError.message }`
- Missing stripe_connected_account: return appropriate `{ account: null }` (not 404)
- Auth failures: 401

**Helper to call stripe-connect Edge Function:**
```typescript
async function callStripeConnectFunction<T>(
  action: string,
  body?: Record<string, unknown>
): Promise<T> {
  const supabase = createClient()
  const { data: sessionData } = await supabase.auth.getSession()
  const token = sessionData.session?.access_token
  if (!token) throw new Error('Not authenticated')

  const baseUrl = process.env.NEXT_PUBLIC_SUPABASE_URL
  const response = await fetch(`${baseUrl}/functions/v1/stripe-connect`, {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${token}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ action, ...body }),
  })

  if (!response.ok) {
    const err = await response.json().catch(() => ({ error: response.statusText }))
    throw new Error(err.error ?? `stripe-connect failed: ${response.status}`)
  }

  return response.json() as Promise<T>
}
```
