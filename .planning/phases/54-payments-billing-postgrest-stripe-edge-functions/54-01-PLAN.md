**useSetDefaultPaymentMethod() mutation:**
Two-step: (1) set all payment_methods for this tenant to is_default=false, (2) set the target to is_default=true.

Step 1: Get tenant record to get tenant_id:
```typescript
const { data: { user } } = await supabase.auth.getUser()
// get tenant id
const { data: tenant } = await supabase
  .from('tenants')
  .select('id')
  .eq('user_id', user.id)
  .single()
// clear all defaults
await supabase
  .from('payment_methods')
  .update({ is_default: false })
  .eq('tenant_id', tenant.id)
// set new default
const { error } = await supabase
  .from('payment_methods')
  .update({ is_default: true })
  .eq('id', paymentMethodId)
if (error) handlePostgrestError(error, 'payment_methods')
```

```typescript
// Get leases owned by this user with units/tenants join
const { data, error } = await supabase
  .from('rent_payments')
  .select('id, amount, currency, status, due_date, tenant_id, lease_id, period_start, period_end')

  .lte('due_date', thirtyDaysOut)
  .eq('status', 'pending')
  .order('due_date')
  .limit(50)

return (data ?? []) as unknown as UpcomingPayment[]
```
