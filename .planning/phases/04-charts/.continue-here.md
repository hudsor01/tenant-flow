## Critical Anti-Patterns (do NOT repeat)

- **Lefthook lint failure on regenerated/dropped JSON files** â€” `gsd-sdk query` writes JSON with spaces; biome project standard is tabs. After any state-setting command that writes JSON to `.planning/config.json` or `.planning/STATE.md`, run `bunx biome format --write {file}` before committing or the pre-commit hook will trip.
- **`bun run test:unit -- --run <file>` double-flag conflict** â€” the `test:unit` script already passes `--run`. Add a SECOND `--run` and vitest crashes with "Expected a single value for option --run, received [true, true]". Use `bunx vitest --run --project unit <file>` directly OR drop the second `--run`.
- **MCP `generate_typescript_types` returns a 90K-char JSON blob** â€” Read tool can't load it.
