  <files>supabase/migrations/20260304150000_financial_fixes_rpcs.sql</files>
  <action>
Create a second migration file with three SECURITY DEFINER RPCs. All must follow Phase 1 patterns: `SET search_path TO 'public'`, validate `auth.uid()`, use `(select auth.uid())`.

1. **PAY-02 + PAY-08: `record_rent_payment` RPC** — Accepts: `p_stripe_payment_intent_id text`, `p_rent_due_id uuid`, `p_tenant_id uuid`, `p_lease_id uuid`, `p_amount numeric(10,2)`, `p_gross_amount numeric(10,2)`, `p_platform_fee_amount numeric(10,2)`, `p_stripe_fee_amount numeric(10,2)`, `p_net_amount numeric(10,2)`, `p_currency text`, `p_period_start text`, `p_period_end text`, `p_due_date text`, `p_checkout_session_id text DEFAULT NULL`. This RPC is called by Edge Functions using service_role key, so it does NOT validate auth.uid() (webhook handler runs as service_role). It: (a) Upserts into `rent_payments` with `ON CONFLICT (stripe_payment_intent_id) DO UPDATE SET status = 'succeeded'`, (b) Updates `rent_due.status` to `'paid'` ONLY when the sum of all succeeded `rent_payments` for that `rent_due_id` >= `rent_due.amount`. This handles shared leases where multiple tenants pay portions. Use the exact pattern from RESEARCH.md Pattern 1.

2. **PAY-16: `set_default_payment_method` RPC** — Accepts: `p_payment_method_id uuid`. Validates that the payment method belongs to a tenant whose `user_id = (select auth.uid())`. In one transaction: clears `is_default` on all payment methods for that tenant, then sets `is_default = true` on the target method. Returns void. RAISE EXCEPTION if method not found or not owned by caller.

3. **PAY-03: `toggle_autopay` RPC** — Accepts: `p_lease_id uuid`, `p_enabled boolean`, `p_payment_method_id text DEFAULT NULL`. Validates caller is a tenant on this lease (join tenants -> lease_tenants). Updates `leases.auto_pay_enabled` and `leases.autopay_payment_method_id`. If disabling, set payment_method_id to null. RAISE EXCEPTION if caller is not a tenant on the lease.

Add GRANT EXECUTE on all three RPCs to `authenticated` role.
